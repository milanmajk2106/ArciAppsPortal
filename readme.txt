# ArciWeb-Portal v1.31

Centrálny portál pre správu nástrojov ArciApps a synchronizáciu hernıch dát (savy).

## Hlavné funkcie
- **Cloud Save/Load**: Ukladanie .txt súborov (do 1MB) do cloudu (Firebase Firestore) so 4 slotmi.
- **Autentifikácia**: Prihlasovanie a registrácia cez Firebase Auth (Email/Heslo).
- **Zabudnuté heslo**: Automatické odosielanie e-mailu na obnovu hesla.
- **Copy Tool**: Rıchle kopírovanie skriptov do schránky.
- **Integrácia**: Priame prepojenia na CMTracker, KlubMerger a StatsKeeper.
